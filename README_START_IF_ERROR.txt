If CRM or API does not open:

1. Do not look at browser first. Open the black window named "Backend API CRM".
2. If backend is installing packages, wait 1-3 minutes.
3. If there is an error, send the first red error or send backend_start_log.txt.
4. Check links:
   Backend health: http://127.0.0.1:8000/api/health
   API docs:       http://127.0.0.1:8000/docs
   CRM:            http://127.0.0.1:8000/crm
   Site:           http://127.0.0.1:5500
5. If port 8000 is busy, close old black windows and run start_project.bat again.
