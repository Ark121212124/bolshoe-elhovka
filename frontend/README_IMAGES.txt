ИНСТРУКЦИЯ ПО ЗАМЕНЕ ФОТОГРАФИЙ И ИКОНОК

1. Герб / иконка сайта в шапке
- Файл: assets/img/icons/site-logo.png
- Сейчас туда уже вставлен герб Большеелховского сельского поселения.
- Чтобы заменить герб, положите новый PNG-файл с таким же названием.
- Картинка автоматически подгоняется по центру и не обрезается.

2. Остальные иконки сайта
- Папка: assets/img/icons/
- Все иконки в этой папке имеют формат PNG.
- Для замены просто сохраните свою иконку с тем же названием и расширением .png.

3. Обычные фотографии
- Главное фото: assets/img/custom/hero-main.jpg
- Фото страниц: assets/img/custom/about-main.jpg, contacts-main.jpg, residents-main.jpg
- Фото новостей: assets/img/news/news-01.jpg, news-02.jpg и далее
- Фото событий: assets/img/events/event-01.jpg, event-02.jpg и далее
- Фото сотрудников: assets/img/staff/
- Фото внутренних разделов: assets/img/sections/

4. Как подгоняются изображения
- Фотографии JPG автоматически заполняют блок и аккуратно обрезаются по центру.
- Иконки PNG и герб автоматически помещаются внутри блока без сильного обрезания.
- Код менять не нужно: достаточно заменить файл на новый с тем же именем.

5. Подложки у иконок
- Круглые и белые подложки вокруг иконок отключены.
- Чтобы иконка была без фона, используйте PNG с прозрачным фоном.
- Если в самой PNG-картинке уже нарисован фон или круг, он останется, поэтому лучше вставлять прозрачные PNG-иконки.

Фотографии внутренних разделов
-----------------------------
Для страниц, которые открываются по кнопке «Открыть раздел», используются файлы из папки:
assets/img/sections/

Главное фото раздела называется по id раздела, например:
assets/img/sections/mkd.jpg
assets/img/sections/bezopasnost.jpg
assets/img/sections/vodosnabzhenie.jpg

Если фотография не найдена, сайт автоматически покажет:
assets/img/sections/section-default.jpg

Также на странице раздела есть дополнительные фотоблоки. Их можно заменить через админ-редактор или вручную в коде/JSON.
