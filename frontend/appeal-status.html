Проверка статуса обращения

Введите номер обращения и телефон, указанный при подаче. Система получит актуальный статус из CRM-модуля.

Как узнать номер обращения?

  • Номер выдается помощником сразу после успешной отправки обращения.
  • Номер имеет вид BR-2026-00001.
  • Для защиты данных дополнительно проверяется телефон заявителя.
  • Если номер потерян, можно обратиться в администрацию по телефону +7 (83441) 3-09-90.
